# Solveur Bolgrot pour Dofus

Cette application web vous permet de résoudre le combat tactique contre le Bolgrot afin de continuer votre Dofus Turquoise.

Renseignez la configuration du combat et l'outil calculera automatiquement la stratégie optimale pour venir à bout de ce redoutable adversaire !

## Fonctionnalités

* Résolution automatique du combat contre le Bolgrot
* Calcul d'une stratégie optimisée tour par tour
* Interface web simple à utiliser
* Aide à la prise de décision pour un combat tactique exigeant

## Utilisation

1. Ouvrez l'application web.
2. Renseignez la configuration actuelle du combat.
3. Lancez la résolution.
4. Suivez les actions proposées par le solveur.

## Objectif

L'objectif de cet outil est d'aider les joueurs à franchir l'étape du combat contre le Bolgrot dans la quête du Dofus Turquoise, sans avoir à tester manuellement des dizaines de possibilités.

## Transparence

⚠️ Vibe codé avec Claude et ChatGPT.

Derrière le vibe coding, ce projet représente plusieurs jours d'analyse, de tests, de validation et d'ajustements manuels sur les mécaniques du combat contre le Bolgrot.
